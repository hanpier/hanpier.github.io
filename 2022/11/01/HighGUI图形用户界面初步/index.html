HighGUI图形用户界面初步

图像的载入、显示和输出到文件

图像的载入：imread()函数

Mat imread(const string& filename, int flags=1);

（1）filename：文件的名字

（2）flags：指定加载图像的颜色类型，默认值为1.

  • CV_LOAD_IMAGE_UNCHANGED——等价取值为-1，读取原图不进行任何改
  • CV_LOAD_IMAGE_GRAYSCALE——等价取值为0，将图像转换成灰度再返回
  • CV_LOAD_IMAGE_COLOR——等价取值为1，转换为彩色再返回。
图像的显示：imshow()函数

void imshow(const string& winname, InputArray mat);

（1）winname：窗口的名字

（2）mat：需要显示的图像

如果图像以CV_WINDOW_AUTOSIZE（默认值）创建，那么显示图像原始大小。否则图像将缩放大小以适应窗口。

  • 如果图像是8位无符号的，它将按原样显示

  • 如果图像是16位无符号或32位整数，像素就除以256。也就是说，值范围[0,255*256]被映射到[0,255]

  • 如果图像是32位或64位浮点数，像素值将乘以255。也就是说，值范围[0,1]被映射到[0,255]。

创建窗口：namedWindow()函数

void nameWind(const string& winname, int flags=WINDOW_AUTOSIZE);

（1）winname：窗口名字，如果相同名字的窗口已经存在，则函数不做任何事情

（2）flags：窗口的标识

  • WINDOW_NORMAL：设置这个值，用户可以改变窗口的大小。
  • WINDOW_AUTOSIZE：设置这个值，窗口大小会自动调整以适应所显示的图像，并且用户不能手动改变窗口大小。
  • WINDOW_OPENGL：设置这个值，窗口创建的时候会支持OPENGL。
输出图像到文件：imwrite()函数

bool imwrite(const string& filename, InputArry img, const vector<int>& params=vector<int>() )

（1）filename：需要写入的文件名

（2）img：Mat类型的图像数据

（3）paras：表示为特定格式保存的参数编码

滑动条的创建和使用

创建滑动条：creatTrackbar()函数

int creatTrackbar(conststring& trackbarname,conststring &winname, int* value, int count, TrackbarCallback onChange=0, void* userdata=0);

（1）trackname：轨迹条的名字，用来代表我们创建的轨迹条。

（2）winname：窗口的名字，可以用namedWindow()创建

（3）value：滑块的位置，在创建时滑块的初始位置就是该变量当前的值

（4）count：滑块能达到的最大位置的值，最小位置的值为0

（5）onChange：指向回调函数的指针，默认值为0，每次滑块位置改变，函数就回回调。函数原型：***void (TrackbarCallback)(int pos, void* userdata)***，第一个参数是轨迹条的位置，第二个是用户传入数据userdata

（6）userdata：用户传给回调函数的数据，用来处理轨迹条事件，默认值为0

获取当前轨迹条的位置：getTrackbarPos()

int getTrackbarPos(conststring& trackbarname, conststring& winname)

（1）trackbarname：轨迹条的名字

（2）winname：轨迹条的父窗口名字

鼠标操作

void setMouseCallback(conststring& winname, MouseCallback onMouse, void* userdata=0)

（1）winname：窗口的名字

（2）onMouse：鼠标事件发生时，被调用的函数指针。函数原型为void Foo(int event, int x, int y, int flags, void*param)

（3）userdata：用户定义的传递到回调函数的参数，默认为0
